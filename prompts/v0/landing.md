# Landing page brief

- Utiliser les composants ShadCN existants (`Hero`, `Benefits`, `Testimonials`, `FooterHero`, `Footer`) comme base.
- Respecter la palette `zinc` et les tokens Tailwind définis dans `globals.css`.
- Prévoir une section héros percutante, une preuve sociale et un appel à l'action clair orienté vers l'inscription.
- Les images statiques se trouvent dans `public/images`. Préférer les images existantes au téléchargement de nouvelles ressources.
- Le code généré doit rester dans `src/app/(marketing)` pour conserver le découpage marketing/app.
